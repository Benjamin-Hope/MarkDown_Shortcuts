## API Reference

#### Get all items

```http
  GET /api/items
```

| Parameter | Type     | Description                                    |
| :-------- | :------- | :--------------------------------------------- |
| `id`      | `string` | **Required**. ID of the item you're requesting |

#### Get item

```http
  POST /api/items
```

| Parameter | Type     | Description                      |
| :-------- | :------- | :------------------------------- |
| `name`    | `string` | **Required**. Name of a new item |
| `price`   | `number` | **Required**. Name of a new item |
